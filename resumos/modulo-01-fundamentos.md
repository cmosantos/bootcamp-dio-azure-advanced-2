# Módulo 1 – Fundamentos de Azure e Computação em Nuvem

## 1. Conceitos de computação em nuvem

- Diferença entre ambiente local (on-premises) e nuvem
- Modelos de serviço:
  - **IaaS** – infraestrutura como serviço
  - **PaaS** – plataforma como serviço
  - **SaaS** – software como serviço
- Modelos de implantação:
  - nuvem pública, privada, híbrida e multicloud

## 2. Plataforma Microsoft Azure

- Regiões e datacenters distribuídos globalmente
- Estrutura lógica:
  - **Tenant (Entra ID)**
  - **Subscription**
  - **Resource Group**
  - **Resources**
- Boas práticas:
  - organizar RGs por ambiente/projeto
  - uso de **tags** para identificar custo, equipe, ambiente etc.

## 3. Portal Azure

- Tela inicial e barra de busca
- Principais blades:
  - Resource groups
  - All resources
  - Subscriptions
  - Cost Management + Billing
- Ideia de “ponto central” para administrar recursos de nuvem

## 4. Responsabilidade compartilhada

- Provedor cuida: datacenter físico, rede, energia, hardware, host
- Cliente cuida: identidade, dados, configuração de serviços, acessos
- Importância de:
  - backup
  - governança
  - controle de custos

## 5. Conexão com a certificação AZ-900

- Conceitos que caem na prova:
  - modelos de nuvem e serviço
  - estrutura do Azure (regiões, RG, assinaturas)
  - responsabilidade compartilhada
  - pilares de segurança, confiabilidade e custos
